

------

每次启动项目，需要按这个顺序：

```bash
cd ~/Desktop/local-rag-system
conda activate RagSystem
```

启动 Qdrant：

```bash
docker compose up -d
```

确认 LM Studio 打开，并且两个模型都 loaded：

```text
qwen/qwen3-v1-8b
text-embedding-bge-small-zh-v1.5
```

启动 FastAPI：

```bash
uvicorn app.main:app --reload
```

测试健康检查：

```bash
curl http://127.0.0.1:8000/health
```

测试上传文档：

```bash
curl -X POST "http://127.0.0.1:8000/documents/upload" \
  -F "file=@项目设计.md"
```

